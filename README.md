* 解递归式的3种方法： 
	* 代换法 猜答案、解出常数、用数学归纳法 
	* 递归树法 把式子分解开，画成树的形状，持续展开，逐层求和 
	* 主定理 T(n)=aT(n/b)+f(n),a>=1,b>1,f(n)渐进趋正 和n^logba比较
		* f(n) > n^logba: Θ(f(n))
		* f(n) = n^logba: Θ(n^logba*lgn)
		* f(n) < n^logba: Θ(n^logba)

* 斐波那契数列:
	* 这个数列从第三项开始，每一项都等于前两项之和
	* Θ(lgn)
	```
	⎡f(n+1) f(n)⎤ = ⎡1 1⎤^n
	⎣f(n) f(n-1)⎦   ⎣1 0⎦
	```

* 斯特拉森矩阵乘法:
	* n*n 矩阵且 n 为2的整数次幂
	* Θ(n^lg7)
	```
	    a11 a12     b11 b12	 
	A = a21 a22 B = b21 b22
	```
	```
	x1 = (a11 + a22) * (b11 + b22);
	x2 = (a21 + a22) * b11;
	x3 = a11 * (b12 - b22);
	x4 = a22 * (b21 - b11);
	x5 = (a11 + a12) * b22;
	x6 = (a21 - a11) * (b11 + b12);
	x7 = (a12 - a22) * (b21 + b22);
	```
	```
	c11 = x1 + x4 - x5 + x7
	c12 = x3 + x5
	c21 = x2 + x4
	c22 = x1 + x3 - x2 + x6
	```
	```
	    c11 c12
	C = c21 c22
	```
